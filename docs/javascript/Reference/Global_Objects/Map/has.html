方法has() 返回一个bool值，用来表明map 中是否存在指定元素.

语法

myMap.has(key);

参数

key
必填. 用来检测是否存在指定元素的键值.

返回值

Boolean
如果指定元素存在于Map中，则返回true。其他情况返回false

案例

使用has方法

var myMap = new Map();
myMap.set("bar", "foo");

myMap.has("bar");  // returns true
myMap.has("baz");  // returns false
