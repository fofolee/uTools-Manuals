getUTCDate() 方法以世界时为标准，返回一个指定的日期对象为一个月中的第几天

语法

dateObj.getUTCDate()

参数

无

返回值

getUTCDate() 返回一个 1 到 31 的整数值

例子

例子: 使用 getUTCDate() 方法

下面的例子是把当前日期的天数部分赋值给变量 day.

var today = new Date();
var day = today.getUTCDate();
