概述

Math.exp() 函数返回 ex，x 表示参数，e 是欧拉常数（Euler's constant），自然对数的底数。

语法

Math.exp(x)

参数

x
一个数值

描述

由于 exp 是 Math 的静态方法，所以应该像这样使用：Math.exp()，而不是作为你创建的 Math 实例的方法。

示例

例子：使用 Math.exp

Math.exp(-1); // 0.36787944117144233
Math.exp(0);  // 1
Math.exp(1);  // 2.718281828459045
