RangeError.prototype 属性表示 RangeError 构造函数的原型。
 
RangeError.prototype 属性的属性特性：
writable false
enumerable false
configurable false

描述

所有  RangeError 的实例都继承自 RangeError.prototype ，所以你可以使用这个属性来为所有的实例添加属性或方法。

属性

RangeError.prototype.constructor
指定了创建实例原型的函数
RangeError.prototype.message
错误信息。尽管 ECMA-262 规定了 RangeError 应该拥有一个 message 属性，但在 SpiderMonkey 中，该属性继承自 Error.prototype.message。
RangeError.prototype.name
错误名字，继承自 Error。
RangeError.prototype.fileName
引起该错误的文件路径，继承自 Error。
RangeError.prototype.lineNumber
引起该错误的行号，继承自 Error。
RangeError.prototype.columnNumber
引起该错误的列号，继承自 Error。
RangeError.prototype.stack
堆栈跟踪记录，继承自 Error。

方法

尽管 RangeError 原型对象自身没有包含任何方法，但是 RangeError 实例却通过原型链继承到了一些方法。
