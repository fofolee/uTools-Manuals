LCASE() 函数

LCASE() 函数把字段的值转换为小写。

SQL LCASE() 语法

 SELECT LCASE(column_name) FROM table_name;

用于 SQL Server 的语法

 SELECT LOWER(column_name) FROM table_name;

SQL LCASE() 实例

下面的 SQL 语句从 "Customers" 表中选取 "CustomerName" 和 "City" 列，并把 "CustomerName" 列的值转换为小写：

实例

 SELECT LCASE(CustomerName) AS Customer, City
FROM Customers;
