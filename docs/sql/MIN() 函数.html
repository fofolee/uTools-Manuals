SQL MIN() Function

MIN() 函数

MIN() 函数返回指定列的最小值。

SQL MIN() 语法

 SELECT MIN(column_name) FROM table_name;

演示数据库

SQL MIN() 实例

下面的 SQL 语句从 "Products" 表的 "Price" 列获取最小值：

实例

 SELECT MIN(Price) AS SmallestOrderPrice FROM Products;
