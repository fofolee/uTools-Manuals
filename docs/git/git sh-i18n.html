名称

git-sh-i18n  -  Git 的 shell 脚本的i18n设置代码

概要

. "$(git --exec-path)/git-sh-i18n"

描述

这不是最终用户想要运行的命令。永远。本文档适用于正在研究Pornic-ish脚本和/或正在编写新脚本的人员。

'git sh-i18n scriptlet 被设计为由 Git 在 shell 脚本中实现的瓷器程序来源（使用.）。它提供了 GNU 包装gettext和eval_gettext通过访问功能的gettext.sh脚本，并提供系统直通无回退的 GNU gettext 。

功能

gettext

目前，虚拟贯穿功能被实现为一种包装printf(1)。将被更高版本中的真正的gettext 实现所取代。

eval_gettext

目前，虚拟贯穿功能实现printf(1)为由 git-sh-i18n {litdd} envsubst [1]助手扩展的变量包装。将被更高版本中的真正的 gettext 实现所取代。
