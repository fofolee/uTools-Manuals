名称

git-var  - 显示一个 Git 逻辑变量

概要

git var ( -l | <variable> )

描述

打印 Git 逻辑变量。

选项

-l

导致逻辑变量被列出。另外，还列出了 Git 配置文件 .git / config 的所有变量。（但是，配置变量列表功能已被弃用，以支持git config -l。）

示例

$ git var GIT_AUTHOR_IDENT

Eric W. Biederman <ebiederm@lnxi.com> 1121223278 -0600

变量

GIT_AUTHOR_IDENT

一段代码的作者。

GIT_COMMITTER_IDENT

将一段代码放入 Git 的人。

GIT_EDITOR

供 Git 命令使用的文本编辑器。该值用于在 shell 中进行解释。例如：~/bin/vi，$SOME_ENVIRONMENT_VARIABLE，"C:\Program Files\Vim\gvim.exe" --nofork。优先顺序是$GIT_EDITOR环境变量，则core.editor配置，那么$VISUAL，然后$EDITOR，然后将默认选择在编译时，这通常是vi。

GIT_PAGER

供 Git 命令使用的文本查看器（例如，less）。该值旨在由 shell 解释。首选项的顺序是$GIT_PAGER环境变量，然后是core.pager配置，然后$PAGER是编译时（通常less）选择的默认值。
