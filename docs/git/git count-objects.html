命名

git-count-objects  - 计算解包的对象数量及其磁盘消耗量

概要

git count-objects [-v] [-H | --human-readable]

描述

这会计算解包对象文件的数量以及它们所占用的磁盘空间，以帮助您决定何时重新打包。

选项

-v   --verbose

更详细的报告：

计数：松散物体的数量

大小：松散对象消耗的磁盘空间，以KiB为单位（除非指定了-H）

in-pack：包装内物品的数量

size-pack：包中使用的磁盘空间，以KiB为单位（除非指定-H）

prune-packable：包中也存在的松散对象的数量。这些对象可以使用修剪git prune-packed。

garbage：对象数据库中的文件数量既不是有效的松散对象也不是有效的包

size-garbage：由垃圾文件占用的磁盘空间，在KiB中（除非指定了-H）

备用：备用对象数据库的绝对路径; 可能会出现多次，每个路径一行。请注意，如果路径包含不可打印的字符，则它可能被双引号包围，并且包含C风格的反斜杠转义序列。

-H   --human-readable
