字符串函数 - strlen

原型：extern int strlen(char *s);

用法：#include <string.h>

功能：计算字符串s的长度

说明：返回s的长度，不包括结束符NULL。

举例：


      // strlen.c
      
      #include <syslib.h>
      #include <string.h>

      main()
      {
        char *s="Golden Global View";
        
        clrscr();
        
        printf("%s has %d chars",s,strlen(s));

        getchar();
        return 0;
      }
