数学函数 - pow10

原型：extern float pow10(float x);

用法：#include <math.h>

功能：计算10的x次幂。

说明：相当于pow(10.0,x)。

举例：


      // pow10.c
      
      #include <syslib.h>
      #include <math.h>

      main()
      {
        clrscr();        // clear screen
        textmode(0x00);  // 6 lines per LCD screen
      
        printf("10^3.2=%f\n",pow10(3.2));
        printf("10^3.2=%f",pow(10,3.2));
        
        getchar();
        return 0;
      }
相关函数：pow
