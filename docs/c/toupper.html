字符函数 - toupper

原型：extern int toupper(int c);

用法：#include <ctype.h>

功能：将字符c转换为大写英文字母

说明：如果c为小写英文字母，则返回对应的大写字母；否则返回原来的值。

举例： 
    // toupper.c
    
    #include <syslib.h>
    #include <ctype.h>

    main()
    {
      char *s="Hello, World!";
      int i;
      
      clrscr();        // clear screen
      printf("%s\n",s);
      for(i=0;i<strlen(s);i++)
      {
        putchar(toupper(s[i]));
      }
      
      getchar();
      return 0;
    }
相关函数：tolower
