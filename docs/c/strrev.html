字符串函数 - strrev

原型：extern char *strrev(char *s);

用法：#include <string.h>

功能：把字符串s的所有字符的顺序颠倒过来（不包括空字符NULL）。

说明：返回指向颠倒顺序后的字符串指针。

举例：


      // strrev.c
      
      #include <syslib.h>
      #include <string.h>

      main()
      {
        char *s="Welcome To Beijing";
        
        clrscr();
        textmode(0x00);  // 6 lines per screen
        
        printf("%s\n%s",s,strrev(strdup(s)));
                

        getchar();
        return 0;
      }
