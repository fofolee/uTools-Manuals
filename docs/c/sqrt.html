数学函数 - sqrt

原型：extern float sqrt(float x);

用法：#include <math.h>

功能：计算x的平方根。

说明：x应大于等于零。

举例： 


    // sqrt.c
    
    #include <syslib.h>
    #include <math.h>

    main()
    {
      clrscr();        // clear screen
      textmode(0x00);  // 6 lines per LCD screen
      
      printf("sqrt(2000)=%f",sqrt(2000.0));
      
      getchar();
      return 0;
    }
