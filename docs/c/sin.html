数学函数 - sin

原型：extern float sin(float x);

用法：#include <math.h>

功能：计算x（弧度表示）的正弦值。

说明：x的值域为[-1.0，1.0]。

举例：

      // sin.c
      
      #include <syslib.h>
      #include <math.h>

      main()
      {
        float x;
        clrscr();        // clear screen
        textmode(0x00);  // 6 lines per LCD screen
      
        x=M_PI/2;        // M_PI=PI=3.14159265..., defined in math.h
        printf("sin(PI/2)=%f",sin(x));
        
        getchar();
        return 0;
      }
