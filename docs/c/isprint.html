字符函数 - isprint

原型：extern int isprint(int c);

用法：#include <ctype.h>

功能：判断字符c是否为可打印字符（含空格）

说明：当c为可打印字符（0x20-0x7e）时，返回非零值，否则返回零。

举例：


      // isprint.c
      
      #include <syslib.h>
      #include <ctype.h>

      main()
      {
        int c;
        
        clrscr();        // clear screen
        c='a';
        printf("%c:%s\n",c,isprint(c)?"yes":"no");
        c=' ';
        printf("%c:%s\n",c,isprint(c)?"yes":"no");
        c=0x7f;
        printf("%c:%s\n",c,isprint(c)?"yes":"no");
        
        getchar();
        return 0;
      }
