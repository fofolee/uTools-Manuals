数学函数 - pow

原型：extern float pow(float x, float y);

用法：#include <math.h>

功能：计算x的y次幂。

说明：x应大于零，返回幂指数的结果。

举例：


      // pow.c
      
      #include <syslib.h>
      #include <math.h>

      main()
      {
        clrscr();        // clear screen
        textmode(0x00);  // 6 lines per LCD screen
      
        printf("4^5=%f",pow(4.,5.));
        
        getchar();
        return 0;
      }
