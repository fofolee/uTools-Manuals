Docker events 命令

docker events : 从服务器获取实时事件

语法

docker events [OPTIONS]

OPTIONS说明：

  • -f ：根据条件过滤事件；

  • --since ：从指定的时间戳后显示所有事件;

  • --until ：流水时间显示到指定的时间为止；

实例

显示docker 2016年7月1日后的所有事件。
