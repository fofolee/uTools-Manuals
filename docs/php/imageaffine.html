imageaffine

(PHP 5 >= 5.5.0, PHP 7)

imageaffine — 返回经过仿射变换后的图像，剪切区域可选

说明

imageaffine ( resource $image , array $affine [, array $clip ] ) : resource

Warning

本函数还未编写文档，仅有参数列表。

参数

image

由图象创建函数(例如imagecreatetruecolor())返回的图象资源。

affine

数组，其中键为 0 至 5 的数字。

clip

数组，其中键为 "x"，"y"，"width" 和 "height"。

返回值

成功则返回仿射变换后的图像， 或者在失败时返回 FALSE.
