fileinode

(PHP 4, PHP 5, PHP 7)

fileinode — 取得文件的 inode

说明

fileinode ( string $filename ) : int

取得文件的 inode。

参数

filename

文件的路径。

返回值

返回文件的 inode 节点号， 或者在失败时返回 FALSE。

范例

Example #1 将某个文件和当前文件的 inode 进行对比 


<?php
$filename = 'index.php';
if (getmyinode() == fileinode($filename)) {
    echo 'You are checking the current file.';
}
?>

错误／异常

失败时抛出E_WARNING警告。

注释

Note: 此函数的结果会被缓存。参见 clearstatcache() 以获得更多细节。

Tip

自 PHP 5.0.0 起, 此函数也用于某些 URL 包装器。请参见 支持的协议和封装协议以获得支持 stat() 系列函数功能的包装器列表。

参见

  • getmyinode() - 获取当前脚本的索引节点（inode）
  • stat() - 给出文件的信息
