openssl_pkey_get_public

(PHP 4 >= 4.2.0, PHP 5, PHP 7)

openssl_pkey_get_public — 从证书中解析公钥，以供使用。

说明

openssl_pkey_get_public ( mixed $certificate ) : resource

openssl_get_publickey() 从 certificate 中解析公钥，供其他函数使用。

参数

certificate

certificate 可以是以下之一：

  1. 一个 X.509 证书资源
  2. 一个file://path/to/file.pem格式的字符串。文件名必须包含一个PEM编码的证书或者密钥(也许二者都有).
  3. 一个 PEM 格式的公钥。

返回值

成功，返回真实的密钥资源标识符，错误，则返回 FALSE
