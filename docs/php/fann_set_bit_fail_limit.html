fann_set_bit_fail_limit

(PECL fann >= 1.0.0)

fann_set_bit_fail_limit — 设置训练期间使用的误差

说明

fann_set_bit_fail_limit ( resource $ann , float $bit_fail_limit ) : bool

设置训练期间使用的误差。

参数

ann

Neural network resource.

bit_fail_limit

误差。

返回值
