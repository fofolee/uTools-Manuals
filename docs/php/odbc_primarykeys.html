odbc_primarykeys

(PHP 4, PHP 5, PHP 7)

odbc_primarykeys — Gets the primary keys for a table

说明

odbc_primarykeys ( resource $connection_id , string $qualifier , string $owner , string $table ) : resource

Returns a result identifier that can be used to fetch the column names that comprise the primary key for a table.

参数

connection_id

ODBC 连接标识符，详见 odbc_connect()。

qualifier

owner

table

返回值

Returns an ODBC result identifier 或者在失败时返回 FALSE.

The result set has the following columns:

  • TABLE_QUALIFIER
  • TABLE_OWNER
  • TABLE_NAME
  • COLUMN_NAME
  • KEY_SEQ
  • PK_NAME
