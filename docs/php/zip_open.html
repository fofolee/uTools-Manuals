zip_open

(PHP 4 >= 4.1.0, PHP 5 >= 5.2.0, PHP 7, PECL zip >= 1.0.0)

zip_open — 打开ZIP存档文件

说明

zip_open ( string $filename ) : resource

打开一个新的ZIP归档文件进行读取。

参数

filename

待打开ZIP归档的文件名。

返回值

成功的时候返回一个资源句柄供函数zip_read() 和 zip_close()后续使用; 如果filename 文件不存在或者出现其他错误，则会返回相应的错误码。

参见

  • zip_read() - 读取ZIP存档文件中下一项
  • zip_close() - 关闭一个ZIP档案文件
