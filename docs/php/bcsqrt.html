bcsqrt

(PHP 4, PHP 5, PHP 7)

bcsqrt — 任意精度数字的二次方根

说明

bcsqrt ( string $operand [, int $scale ] ) : string

返回操作数的二次方根.

参数

operand

字符串类型的操作数.

scale

此可选参数用于设置结果中小数点后的小数位数。也可通过使用 bcscale() 来设置全局默认的小数位数，用于所有函数。

返回值

返回二次方根的结果为字符串类型，如果操作数是负数则返回null.

范例

Example #1 bcsqrt() 示例


<?php

echo bcsqrt('2', 3); // 1.414

?>
