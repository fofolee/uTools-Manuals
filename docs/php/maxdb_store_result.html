maxdb_store_result

maxdb::store_result

(PECL maxdb >= 1.0)

maxdb_store_result -- maxdb::store_result — Transfers a result set from the last query

说明

过程化风格

maxdb_store_result ( resource $link ) : resource

面向对象风格

maxdb::store_result ( void ) : object

This function has no functionally effect.

返回值

Returns a result resource or FALSE if an error occurred.

范例

See maxdb_multi_query().

参见

  • maxdb_real_query() - Execute an SQL query
  • maxdb_use_result() - Initiate a result set retrieval
