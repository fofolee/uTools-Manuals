imap_bodystruct

(PHP 4, PHP 5, PHP 7)

imap_bodystruct — Read the structure of a specified body section of a specific message

说明

imap_bodystruct ( resource $imap_stream , int $msg_number , string $section ) : object

Read the structure of a specified body section of a specific message.

参数

imap_stream

由 imap_open() 返回的 IMAP 流。

msg_number

The message number

section

The body section to read

返回值

Returns the information in an object, for a detailed description of the object structure and properties see imap_fetchstructure().

参见

  • imap_fetchstructure() - Read the structure of a particular message
