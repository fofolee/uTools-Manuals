deferred.progress()

所属分类：延迟对象 英文文档：deferred.progress()

deferred.progress( progressCallbacks )返回: Deferred

描述: 当Deferred（延迟）对象生成进度通知时，调用添加处理程序。

  • 添加的版本: 1.7deferred.progress( progressCallbacks )

    • progressCallbacks
      类型: Function()
      一个函数，或者函数数组，当Deferred（延迟）对象生成进度通知时被调用的一个函数。

这个参数可以是一个单一的函数或函数数组。 当通过调用 notify或notifyWith 使延迟对象产生进度通知时，progressCallbacks 就会被调用。由于 deferred.progress()返回的是延迟对象，所以其它延迟对象方法可以链接到该对象上。当延迟对象被 resolved 或 rejected 时，进度回调函数将不再被调用。欲了解更多信息，请参阅文档Deferred object。
