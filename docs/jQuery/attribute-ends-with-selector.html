Attribute Ends With Selector [name$="value"]

所属分类：选择器 > 属性选择器 英文文档：Attribute Ends With Selector [name$="value"]

attributeEndsWith selector

描述: 选择指定属性是以给定值结尾的元素。这个比较是区分大小写的。

  • 添加的版本: 1.0jQuery( "[attribute$='value']" )

    attribute: 一个属性名.

    value: 一个属性值，可以是一个不带引号的一个单词或带一个引号的字符串。

例子:

查找所有的属性名称以“letter”的结束，并把他们的文字输入。
