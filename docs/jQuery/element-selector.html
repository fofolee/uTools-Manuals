element selector

描述: 根据给定（html）标记名称选择所有的元素。

  • 添加的版本: 1.0jQuery( "element" )

    element: 一个用来搜索的元素。 指DOM节点的标签名。

调用JavaScript的 getElementsByTagName()函数，当该表达式使用时返回相应的元素。
