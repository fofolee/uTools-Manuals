only-child selector

描述: 如果某个元素是其父元素的唯一子元素，那么它就会被选中。

  • 添加的版本: 1.1.4jQuery( ":only-child" )

若父元素有其他子元素，就不会被匹配。

例子:

若每个 div 中只有一个按钮，则改变这些按钮的外观。
