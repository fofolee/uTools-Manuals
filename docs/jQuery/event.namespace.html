event.namespace

所属分类：事件 > 事件对象 英文文档：event.namespace

event.namespace返回: String

描述: 当事件被触发时此属性包含指定的命名空间。

  • 添加的版本: 1.4.3event.namespace

这属性主要提供给插件作者用来处理的不同任务，具体取决于事件使用的命名空间。
