.click()

所属分类：事件 > 鼠标事件 英文文档：.click()

.click( handler(eventObject) )返回: jQuery

描述: 为 JavaScript 的"click" 事件绑定一个处理器，或者触发元素上的 "click" 事件。

  • 添加的版本: 1.0.click( handler(eventObject) )

    • handler(eventObject)
      类型: Function()
      每次事件触发时会执行的函数。

  • 添加的版本: 1.4.3.click( [eventData ], handler(eventObject) )

    • eventData
      类型: Object
      一个对象，它包含的数据键值对映射将被传递给事件处理程序。
    • handler(eventObject)
      类型: Function()
      每次事件触发时会执行的函数。

  • 添加的版本: 1.0.click()

    • 这个方法不接受任何参数

这个函数的前两个用法是 .on('click', handler) 的快捷方式，第三个用法是.trigger( "click" )的快捷方式。一个元素被点击和鼠标按键被按下和释放时候将触发 click 事件。任何HTML元素都可以接收此事件。举例来说，请看下面的HTML：

For example, consider the HTML:
<div id="target">
  Click here
</div>
<div id="other">
  Trigger the handler
</div>

这个事件处理器可以绑定到任何 <div>:

1
2
3
 
$("#target").click(function() {
  alert("Handler for .click() called.");
});

现在，如果我们点击这个元素，警报提示框显示：

Handler for .click() called.

我们也可以通过点击不同的元素时手动触发这个事件：

1
2
3
 
$("#other").click(function() {
  $("#target").click();
});

这些代码执行后，点击Trigger the handler同样警报提示框显示。

click事件触发需要以下几点：

  • 鼠标指针在元素里面时点击。
  • 鼠标指针在元素里面时释放。

这通常要求的顺序,然后才采取行动。如果这些不被满足，mousedown 或 mouseup事件可能更加合适。
