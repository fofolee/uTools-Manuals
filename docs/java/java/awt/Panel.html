    public class Panel
extends Container
implements Accessible
    Panel是最简单的容器类。 面板提供应用程序可以附加任何其他组件（包括其他面板）的空间。

    面板的默认布局管理器是FlowLayout布局管理器。

    从以下版本开始：
    JDK1.0
    另请参见：
    FlowLayout ， Serialized Form
