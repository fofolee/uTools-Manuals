    Event是一个独立于平台的类，用于在Java 1.0事件模型中封装来自平台的图形用户界面的事件。 在Java 1.1和更高版本中， Event类仅保留向后兼容性。 提供了此类描述中的信息，以帮助程序员将Java 1.0程序转换为新的事件模型。

    在Java 1.0事件模型中，一个事件包含一个id字段，指示它是什么类型的事件，哪个其他Event变量与事件相关。

    对于键盘事件， key包含一个值，指示哪个键已激活，并且modifiers包含该事件的修饰符。 对于KEY_PRESS和KEY_RELEASE事件id的值key是关键的Unicode字符代码。 对于KEY_ACTION和KEY_ACTION_RELEASE，的值key是在所定义的动作密钥标识符之一Event类（ PGUP ， PGDN ， F1 ， F2 ，等）。
