    public class PaintEvent
extends ComponentEvent
    组件级paint事件。 此事件是一种特殊类型，用于确保绘画/更新方法调用与从事件队列传递的其他事件一起被序列化。 此事件不适用于事件监听器模型; 程序应该继续覆盖油漆/更新方法，以便正确地渲染它们。

    如果未指定的行为将引起id任何特定的参数PaintEvent实例不在范围从PAINT_FIRST到PAINT_LAST 。
