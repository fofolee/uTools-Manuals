      • forName

        public static final Character.UnicodeBlock forName(String blockName)
        返回具有给定名称的UnicodeBlock。 块名称由Unicode标准确定。 文件Blocks- <version> .txt为特定版本的标准定义块。 Character类指定其支持的标准版本。

        此方法接受以下形式的块名称：

        1. 规范块名称由Unicode标准定义。 例如，该标准定义了一个“基本拉丁文”块。 因此，该方法接受“基本拉丁文”作为有效的块名称。 每个UnicodeBlock的文档提供了规范名称。
        2. 所有空间被删除的规范块名称。 例如，“BasicLatin”是“基本拉丁”块的有效块名称。
        3. 每个常量UnicodeBlock标识符的文本表示。 例如，如果提供“BASIC_LATIN”名称，此方法将返回BASIC_LATIN块。 此表单将使用下划线替换规范名称中的所有空格和连字符。
        最后，对于所有有效的块名称形式，字符大小写将被忽略。 例如，“BASIC_LATIN”和“basic_latin”都是有效的块名称。 en_US语言环境的案例映射规则用于为块名称验证提供不区分大小写的字符串比较。

        如果Unicode标准更改块名称，则将接受上一个和当前的名称。

        参数
        blockName - A UnicodeBlock名称。
        结果
        该 UnicodeBlock通过鉴定实例 blockName
        异常
        IllegalArgumentException - 如果 blockName是无效的名称
        NullPointerException - 如果 blockName为空
        从以下版本开始：
        1.5
