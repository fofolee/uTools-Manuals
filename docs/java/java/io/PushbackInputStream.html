    public class PushbackInputStream
extends FilterInputStream
    A PushbackInputStream将功能添加到另一个输入流，即可以“推回”或“未读”一个字节。 这在代码片段方便读取由特定字节值分隔的无限数量的数据字节的情况下非常有用; 读取终止字节后，代码段可以“未读”，以便输入流上的下一个读操作将重新读取被推回的字节。 例如，表示构成标识符的字符的字节可以被表示操作符的字节终止; 一个只能读取一个标识符的工作方法可以读取，直到它看到操作员，然后将操作员推回以重新读取。
    从以下版本开始：
    JDK1.0
