    public class SQLTransactionRollbackException
extends SQLTransientException
    当SQLState类值为“ 40 ”时，或在供应商指定的条件下，抛出SQLException的子类。 这表示由于死锁或其他事务序列化故障，当前语句被数据库自动回滚。

    请咨询您的驱动程序供应商文档，了解供应商异常可能抛出此异常 。

    从以下版本开始：
    1.6
