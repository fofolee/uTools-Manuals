    public enum SignStyle

    列举处理正/负号的方法。

    格式化引擎允许使用此枚举来控制数字的正数和负数。 参见DateTimeFormatterBuilder的使用。

    实现要求：
    这是一个不可变的和线程安全的枚举。
    从以下版本开始：
    1.8
