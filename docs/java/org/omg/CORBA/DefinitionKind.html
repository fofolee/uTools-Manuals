Class DefinitionKind

  • java.lang.Object
    • org.omg.CORBA.DefinitionKind
  • All Implemented Interfaces:
    Serializable ， IDLEntity 

    public class DefinitionKind
extends Object
implements IDLEntity
    提供用于标识Interface Repository对象类型的常量的类。 这个类包含两种常量，那些是int和那些是DefinitionKind类的实例。 该类提供方法from_int ，其中给出了一个int常量，创建了相应的DefinitionKind实例。 它还提供方法value ，它返回int常数，这是DefinitionKind实例的值。
