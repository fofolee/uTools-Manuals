    public interface Attribute
extends Serializable
    接口属性是由任何和每个打印属性类实现的基本接口，以指示该类表示打印属性。 所有打印属性都是可序列化的。

    • 方法详细信息

      • getCategory

        类<? extends Attribute> getCategory()
        当打印属性值添加到属性集中时，获取打印属性类，该属性类将用作此打印属性值的“类别”。
        结果
        打印属性类（category），一个类别为java.lang.Class的实例。

      • getName

        String getName()
        获取此属性值是一个实例的类别的名称。

        注意：此方法旨在为属性的类别提供一个默认的非本地化字符串。 如果两个属性对象从返回同一类别getCategory()方法，他们应该从返回的同名getName()方法。

        结果
        属性类别名称。
