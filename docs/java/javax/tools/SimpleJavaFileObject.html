    public class SimpleJavaFileObject

    为JavaFileObject中的大多数方法提供简单的实现。 该类被设计为子类化，并用作JavaFileObject实现的基础。 只要符合JavaFileObject的一般合约，子类就可以覆盖此类的任何方法的实现和规范。
