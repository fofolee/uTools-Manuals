Package javax.swing.event Description

提供Swing组件触发的事件。 除了java.awt.event包中的这些事件之外，它还包含Swing组件触发的事件的事件类和事件侦听器接口。

注意：大多数Swing API 不是线程安全的。 有关详细信息，请参阅Threads and Swing ，在部分The Java Tutorial 。
