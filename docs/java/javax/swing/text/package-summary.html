Package javax.swing.text Description

提供处理可编辑和不可编辑文本组件的类和接口。 文本组件的示例是文本字段和文本区域，其中密码字段和文档编辑器是特殊的实例。 此软件包支持的功能包括选择/突出显示，编辑，样式和键映射。

注意：大多数Swing API 不是线程安全的。 有关详细信息，请参阅Threads and Swing ，在部分The Java Tutorial 。
