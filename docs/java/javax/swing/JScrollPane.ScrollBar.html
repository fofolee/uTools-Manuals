    protected class JScrollPane.ScrollBar
extends JScrollBar
implements UIResource
    默认情况下， JScrollPane创建作为此类的实例的滚动条。 Scrollbar将getUnitIncrement和getBlockIncrement方法，以便如果视口的视图为Scrollable ，则会要求该视图计算这些值。 除非明确设置单位/块增量。

    警告：此类的序列化对象与将来的Swing版本不兼容。 当前的序列化支持适用于运行相同版本的Swing的应用程序之间的短期存储或RMI。 从1.4开始，对所有JavaBeans的长期存储的支持已经添加到java.beans包中。 请参阅XMLEncoder 。

    另请参见：
    Scrollable ， JScrollPane.createVerticalScrollBar() ， JScrollPane.createHorizontalScrollBar()
