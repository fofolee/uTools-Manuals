    public class DefaultTreeCellEditor.DefaultTextField
extends JTextField
    TextField当没有编辑器提供时使用。 这个文本框锁定到它构造的边框。 它也喜欢它的字体的父母字体。 如果渲染器不是null ，并且没有指定字体，那么首选高度是渲染器的高度。
