    public class BasicComboPopup
extends JPopupMenu
implements ComboPopup
    这是一个基本的贯彻ComboPopup接口。 该类代表组合框的弹出部分的ui。

    所有事件处理由使用createxxxListener()方法和内部类创建的侦听器类来处理。 您可以通过覆盖createxxxListener()方法并提供自己的事件侦听器或从此类中提供的子类化来更改此类的行为。

    警告：此类的序列化对象与将来的Swing版本不兼容。 当前的序列化支持适用于运行相同版本的Swing的应用程序之间的短期存储或RMI。 从1.4开始，对所有JavaBeans的长期存储的支持已经添加到java.beans包中。 请参阅XMLEncoder 。
