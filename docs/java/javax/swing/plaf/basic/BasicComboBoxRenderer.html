      • getListCellRendererComponent

        public Component getListCellRendererComponent(JList list,
                                              Object value,
                                              int index,
                                              boolean isSelected,
                                              boolean cellHasFocus)
        说明从界面： ListCellRenderer复制
        返回已配置为显示指定值的组件。 然后调用该组件的paint方法来“渲染”单元。 如果需要计算列表的维度，因为列表单元格不具有固定的大小，则调用此方法来生成可调用getPreferredSize 。
        Specified by:
        getListCellRendererComponent在界面 ListCellRenderer
        参数
        list - 我们正在画的JList
        value - 由list.getModel（）返回的值getElementAt（index）。
        index - 细胞指数。
        isSelected - 如果指定的单元格被选中，则为真。
        cellHasFocus - 如果指定的单元格具有焦点，则为真。
        结果
        其paint（）方法将呈现指定值的组件。
        另请参见：
        JList ， ListSelectionModel ， ListModel
