    public class JRadioButton
extends JToggleButton
implements Accessible
    单选按钮的实现 - 可以选择或取消选择的项目，并将其状态显示给用户。 与ButtonGroup对象一起使用来创建一组按钮，其中一次只能选择一个按钮。 （创建一个ButtonGroup对象，并使用其add方法在组中包含JRadioButton对象。）
    Note: The ButtonGroup object is a logical grouping -- not a physical grouping. To create a button panel, you should still create a JPanel or similar container-object and add a Border to it to set it off from surrounding components.

    按钮可以配置，并在一定程度上由Action秒控制。 使用带按钮的Action除了直接配置按钮之外，还有很多好处。 请参阅Swing Components Supporting Action了解更多详情，您可以找到更多信息How to Use Actions ，在Java教程的部分。

    有关更多文档 ，请参阅Java教程中的How to Use Buttons, Check Boxes, and Radio Buttons。

    警告： Swing不是线程安全的。 有关更多信息，请参阅Swing's Threading Policy 。

    警告：此类的序列化对象与将来的Swing版本不兼容。 当前的序列化支持适用于运行相同版本的Swing的应用程序之间的短期存储或RMI。 从1.4开始，对所有JavaBeans的长期存储的支持已经添加到java.beans包中。 请参阅XMLEncoder 。

    ButtonGroup ， JCheckBox
