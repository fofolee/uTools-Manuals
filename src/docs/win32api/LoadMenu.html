函数功能

该函数从与应用事例相联系的可执行文件（．EXE）中加载指定的菜单资源。

函数原型

HMENU LoadMenu（HINSTANCE hlnstance，LPCTSTR lpMenuName）；

参数

hlnstance：含有被加载菜单资源的事例模块的句柄。
LpMenuName：指向含有菜单资源名的以空结束的字符串的指针。同时，此参数可由低位字上的资源标识符和高位字上的零组成。要创建此值，用MAKEINTRESOURCE宏。

返回值

如果函数调用成功，返回值是菜单资源句柄；如果函数调用失败，返回值是NULL。若想获得更多的错误信息，请调用GetLastError函数。
备注：关闭应用程序之前，用函数DestroyMenu来销毁菜单并释放加载菜单占用的内存。Windows CE 1.0不支持层叠式菜单。Windows CE 2.0及更高版本支持层叠式菜单。

速查

Windows NT：3.1及以上版本；Windows：95及以上版本；Windows CE：不支持；头文件：winuser.h；输入库：user32.lib；Unicode：在Windows NT环境下，以Unicode和ANSI方式实现。
