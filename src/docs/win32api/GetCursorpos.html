函数功能

该函数检取光标的位置，以屏幕坐标表示。

函数原型

BOOL GetCursorPos（LPPOlNT IpPoint）；

参数

IpPint：POINT结构指针，该结构接收光标的屏幕坐标。

返回值

如果成功，返回值非零；如果失败，返回值为零。若想获得更多错误信息，请调用GetLastError函数。
备注：光标的位置通常以屏幕坐标的形式给出，它并不受包含该光标的窗口的映射模式的影响。该调用过程必须具有对窗口站的WINSTA_READATTRIBUTES访问权限。

速查

Windows NT：3.1及以上版本：Windows：95及以上版本；Windows CE：不支持；头文件：winuser.h；库文件：user32.lib。
