函数功能

该函数是应用程序定义的回调函数，为EnumObjects函数一起使用。此函数用来处理对象数据，GOBJECTENUMPROC类型定义了指向该回调函数的指针，EnumObjectsProc是应用程序定义函数名字的一个占位符。

函数原型

VOID CALLBACK EnumObjectsProc(PVOID lpLogObject, LPARAM lpData)；

参数

lpLogObject：指向描述对象特性的LOGPEN或LOGBRUSH结构的指针。
lpData：指向应用程序定义的数据的指针，该数据由EnumObjects函数传送。

返回值

无。
注释：应用程序必须通过传送函数的地址给EnumObjects函数来注册该函数。

速查

Windows NT：3.1及以上版本；Windows：95及以上版本；Windows CE：不支持；头文件：wingdi.h；库文件：用户自定义。
