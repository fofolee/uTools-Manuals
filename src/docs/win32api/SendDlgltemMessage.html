函数功能

该函数把一个消息发送给指定的对话框中的控制。

函数原型

LONG SendDlgltemMessage(HWND hDlg,int nlDDlgltem,UINT Msg,WPARAM wParam,LPARAM IParam);

参数

hDlg:指定含有控制的对话框。
nlDDigltem:指定接收消息的控制的标识符。
Msg：指定将被发送的消息。
wParam:指定消息特定的其他信息。
IParam:指定消息特定的其他信息。

返回值

返回值指定消息处理的结果，且依赖于发送的消息。
备注：SendDlgltemMessage函数直到消息已经被处理时才返回。
使用SendDlgltemMessage函数同从一个指定的控制中检索句柄和调用SendMessagge函数一样。

速查

Windows NT：3.1及以上版本；Windows：95及以上版本；Windows CE：1.0及以上版本；头文件：winuser.h；库文件：user32.lib Unicode：在Windows NT上实现为Unicode和ANSI两种版本。
