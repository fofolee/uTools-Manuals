函数功能

该函数取得指定菜单项的边界矩形。

函数原型

BOOL GetMenultemRect（HWND hWnd，HMENU hMenu，UINT ultem,LPRECT lprcltem）；

参数

hWnd：含有指定菜单的窗口的句柄。
在Windows NT和Windows 98环境中，如果此值为NULL且hMenu代表一个弹出式菜单，此函数将找到菜单窗口。
ultem：菜单项相对于零的位置。
lprcltem：指向结构RECT的指针，该结构接收指定菜单项的边界矩形（按屏幕坐标）。

返回值

如果函数调用成功，返回非零值；如果函数调用失败，返回值是零。若想获得更多的错误信息，请调用GetLastError函数。

速查

Windows NT：4.0及以上版本；Windows：95及以上版本；Windows CE：1.0及以上版本；头文件：winuser.h;输入库：user32.lib。
