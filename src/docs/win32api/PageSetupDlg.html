函数功能

该函数创建一个PageSetup对话框，此对话框能使用户指定打印页的属性。这些属性包括纸张大小和来源，送纸方向和页边距。

函数原型

BOOL PageSetupDlg（LPPAGESETUPDLGI ppsd）；

参数

Ippsd：指向一个包含初始化对话框信息的PAGESETUPDLG结构。当函数返回时，该结构存放有关用户选择的信息。

返回值

如果用户点击OK钮，返回值为非零值，Ippsp参数指向的PAGESETUPDLG结构中的成员显示用户的选择。如果用户取消或关闭PageSetup对话框或错误出现，返回值为零。若想获得更多的错误信息，请调用CommDlgExtendedError函数

速查

Windows NT：3.1及以上版本；Windows：95及以上版本；Windows CE：不支持；头文件：commdlg.h；库文件：commdlg32.lib；Unicode：在Windows NT环境中实现为Unicode和ANSI两个版本。
