函数功能

该函数创建一个Save公共对话框，以便用户指定驱动器、目录和文件名。

函数原型

BOOL GetSaveFileName（LPOPENFILENAME Ipofn）；

参数

Ipofn：指向一个包含初始化对话框信息的OPENFILENAME结构。当GetsaveFileName函数返回时，此结构含有关于用户文件选择的信息。

返回值

如果用户指定了一个文件名且点击OK按钮，返回值为非零值。由OPENFILENAME结构中的归IpstrFile成员指向的缓冲区含有全路径和用户指定的文件名。如果用户取消或关闭Save对话框或错误出现，返回值为零。若想获得有关更多的错误信息，请调用CommDlgExtendError函数，其返回值同GetOpenFileName返回值。

速查

Windows NT：3.1引及以上版本；Windows：95及以上版本；Windows CE：1.0及以上版本；头文件：commdlg.h；库文件：comdlg32.lib；Unicode：在Windows NT环境中实现为Unicode和ANSI两个版本。
