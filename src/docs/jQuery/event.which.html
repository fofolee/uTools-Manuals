event.which

所属分类：事件 > 事件对象 英文文档：event.which

event.which返回: Number

描述: 针对键盘和鼠标事件，这个属性能确定你到底按的是哪个键。

  • 添加的版本: 1.1.3event.which

event.which 将 event.keyCode 和 event.charCode 标准化了。推荐用 event.which 来监视键盘输入。更多细节请参阅： event.charCode on the MDC.

event.which也将正常化的按钮按下(mousedown 和 mouseupevents)，左键报告1，中间键报告2，右键报告3。使用event.which代替event.button。
