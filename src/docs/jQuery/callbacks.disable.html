callbacks.disable()

所属分类：回调对象 英文文档：callbacks.disable()

callbacks.disable()返回: Callbacks

描述: 禁用回调列表中的回调

  • 添加的版本: 1.7callbacks.disable()

    • 这个方法不接受任何参数

此方法返回绑定它的那个回调对象(this).

Example

使用 callbacks.disable() 禁止回调列表中的回调进一步的调用:

// a sample logging function to be added to a callbacks list
var foo = function( value ) {
  console.log( value );
};
 
var callbacks = $.Callbacks();
 
// add the above function to the list
callbacks.add( foo );
 
// fire the items on the list
callbacks.fire( "foo" );
// outputs: foo
 
// disable further calls being possible
callbacks.disable();
 
// attempt to fire with "foobar" as an argument
callbacks.fire( "foobar" );
// foobar isn't output
