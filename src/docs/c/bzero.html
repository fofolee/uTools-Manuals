字符串函数 - bzero

原型：extern void bzero(void *s, int n);

用法：#include <string.h>

功能：置字节字符串s的前n个字节为零。

说明：bzero无返回值。

举例：


      // bzero.c
      
      #include <syslib.h>
      #include <string.h>

      main()
      {
        struct
        {
          int a;
          char s[5];
          float f;
        } tt;
        
        char s[20];
        
        bzero(&tt,sizeof(tt));  // struct initialization to zero
        bzero(s,20);
        
        clrscr();
        printf("Initail Success");

        getchar();
        return 0;
      }
