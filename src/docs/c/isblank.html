字符函数 - isblank

原型：extern int isblank(int c);

用法：#include <ctype.h>

功能：判断字符c是否为TAB或空格

说明：当c为TAB或空格时，返回非零值，否则返回零。

举例：


      // isalnum.c
      
      #include <syslib.h>
      #include <ctype.h>

      main()
      {
        int c;
        
        clrscr();        // clear screen
        c='a';
        printf("%c:%s\n",c,isalnum(c)?"yes":"no");
        c='7';
        printf("%c:%s\n",c,isalnum(c)?"yes":"no");
        c='@';
        printf("%c:%s\n",c,isalnum(c)?"yes":"no");
      }
相关函数：isalnum,isalpha,iscntrl,isgraph,isprint,ispunct,isspace
