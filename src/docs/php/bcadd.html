bcadd

(PHP 4, PHP 5, PHP 7)

bcadd — 2个任意精度数字的加法计算

说明

bcadd ( string $left_operand , string $right_operand [, int $scale ] ) : string

左操作数和右操作数求和

参数

left_operand

左操作数，字符串类型

right_operand

右操作数，字符串类型

scale

此可选参数用于设置结果中小数点后的小数位数。也可通过使用 bcscale() 来设置全局默认的小数位数，用于所有函数。

返回值

2个操作数求和之后的结果以字符串返回

范例

Example #1 bcadd() 示例


<?php

$a = '1.234';
$b = '5';

echo bcadd($a, $b);     // 6
echo bcadd($a, $b, 4);  // 6.2340

?>
