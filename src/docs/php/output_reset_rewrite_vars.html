output_reset_rewrite_vars

(PHP 4 >= 4.3.0, PHP 5, PHP 7)

output_reset_rewrite_vars — 重设URL重写器的值（Reset URL rewriter values）

说明

output_reset_rewrite_vars ( void ) : bool

此函数重置URL重写器，移除所有的先前由 output_add_rewrite_var() 函数设置的重写变量，或者移除会话机制（如果session.use_trans_sid 在 session_start()上进行了设置）。

返回值

成功时返回 TRUE， 或者在失败时返回 FALSE。

范例

Example #1 output_reset_rewrite_vars() example


<?php
session_start();
output_add_rewrite_var('var', 'value');

echo '<a href="file.php">link</a>';
ob_flush();

output_reset_rewrite_vars();
echo '<a href="file.php">link</a>';
?>

以上例程会输出：

<a href="file.php?PHPSESSID=xxx&var=value">link</a>
<a href="file.php">link</a>

参见

  • output_add_rewrite_var() - 添加URL重写器的值（Add URL rewriter values）
  • ob_flush() - 冲刷出（送出）输出缓冲区中的内容
  • ob_list_handlers() - 列出所有使用中的输出处理程序。
  • session_start() - 启动新会话或者重用现有会话
