ob_get_length

(PHP 4 >= 4.0.2, PHP 5, PHP 7)

ob_get_length — 返回输出缓冲区内容的长度

说明

ob_get_length ( void ) : int

此函数将返回输出缓中冲区内容的长度。

返回值

返回输出缓冲区内容的长度；或者返回FALSE——如果没有起作用的缓冲区。

范例

Example #1 A simple ob_get_length() example


<?php

ob_start();

echo "Hello ";

$len1 = ob_get_length();

echo "World";

$len2 = ob_get_length();

ob_end_clean();

echo $len1 . ", ." . $len2;
?>

以上例程会输出：

6, 11
