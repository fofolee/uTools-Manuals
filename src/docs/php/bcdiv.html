bcdiv

(PHP 4, PHP 5, PHP 7)

bcdiv — 2个任意精度的数字除法计算

说明

bcdiv ( string $left_operand , string $right_operand [, int $scale = int ] ) : string

左操作数除以右操作数

参数

left_operand

左操作数，字符串类型

right_operand

右操作数，字符串类型

scale

此可选参数用于设置结果中小数点后的小数位数。也可通过使用 bcscale() 来设置全局默认的小数位数，用于所有函数。

返回值

返回结果为字符串类型的结果，如果右操作数是0结果为null

范例

Example #1 bcdiv() 示例


<?php

echo bcdiv('105', '6.55957', 3);  // 16.007

?>
