seaslog_get_author

(PECL seaslog >=1.0.0)

seaslog_get_author — 获取 SeasLog 作者

说明

seaslog_get_author ( void ) : string

参数

此函数没有参数。

返回值

返回一个 SeasLog 作者的字符串

范例

Example #1 seaslog_get_author() example


<?php

var_dump(seaslog_get_author());

?>

以上例程的输出类似于：

string(29) "Chitao.Gao  [ neeke@php.net ]"
