gupnp_service_introspection_get_state_variable

(PECL gupnp >= 0.1.0)

gupnp_service_introspection_get_state_variable — Returns the state variable data

说明

gupnp_service_introspection_get_state_variable ( resource $introspection , string $variable_name ) : array

Returns the state variable data by the name variable_name in this service.

参数

introspection

A introspection identifier.

variable_name

The name of the variable to retreive.

返回值

Return the state variable data or FALSE.
