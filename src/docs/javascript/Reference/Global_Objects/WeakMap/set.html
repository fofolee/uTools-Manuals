set() 方法根据指定的key和value在 WeakMap对象中添加新/更新元素。

语法

wm.set(key, value);

参数

key
必须的。是要在WeakMap 对象中添加/更新元素的key部分。
value
必须的。是要在WeakMap 对象中添加/更新元素的value部分。

返回值

该WeakMap对象

例子

使用set方法

var wm = new WeakMap();
var obj = {};

// Add new elements to the WeakMap
wm.set(obj, "foo").set(window, "bar"); // chainable

// Update an element in the WeakMap
wm.set(obj, "baz");
