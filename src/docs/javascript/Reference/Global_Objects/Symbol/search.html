Symbol.search 指定了一个搜索方法，这个方法接受用户输入的正则表达式，返回该正则表达式在字符串中匹配到的下标，这个方法由以下的方法来调用 String.prototype.search()。

更多信息请参见 RegExp.prototype[@@search]() 和String.prototype.search().

Symbol.search 属性的属性特性：
writable false
enumerable false
configurable false
