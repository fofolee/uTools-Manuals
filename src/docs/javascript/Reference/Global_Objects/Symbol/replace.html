Symbol.replace 这个属性指定了当一个字符串替换所匹配字符串时所调用的方法。String.prototype.replace() 方法会调用此方法。

更多信息， 详见 RegExp.prototype[@@replace]() 和 String.prototype.replace()。

Symbol.replace 属性的属性特性：
writable false
enumerable false
configurable false
