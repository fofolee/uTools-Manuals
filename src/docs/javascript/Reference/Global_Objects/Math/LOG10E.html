概述

Math.LOG10E 属性表示以 10 为底数，e 的对数，约为 0.434：

Math.LOG10E=log10(e)≈0.434\mathtt{\mi{Math.LOG10E}} = \log_10(e) \approx 0.434

Math.LOG10E 属性的属性特性：
writable false
enumerable false
configurable false

描述

由于 LOG10E 是 Math 的静态属性，所以应该像这样使用：Math.LOG10E，而不是作为你创建的 Math 实例的属性（Math 不是一个构造函数）。

示例

例子：使用 Math.LOG10E

下面的函数返回以 10为底数，E的对数：

function getLog10e() {
   return Math.LOG10E
}

getLog10e() // 0.4342944819032518
