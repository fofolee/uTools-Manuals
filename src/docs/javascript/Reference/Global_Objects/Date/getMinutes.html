getMinutes() 方法根据本地时间，返回一个指定的日期对象的分钟数。

语法

dateObj.getMinutes()

参数

无

描述

getMinutes 返回一个0 到 59的整数值。

例子

例子：使用getMinutes方法

下例中，第二行语句运行过后，变量 minutes 的值为15，也就是说 Xmas95 这个日期对象的值为某时15分某秒。

var Xmas95 = new Date("December 25, 1995 23:15:00");
var minutes = Xmas95.getMinutes();
