信息

SyntaxError: missing } after property list

错误类型

SyntaxError

哪里出错了?

在对象初始化的时候语法出错了。可能是遗漏了一个大括号，也可能是遗漏了逗号。还要检查是否以正确的顺序关闭了大括号或括号。 缩进或格式化代码也许可以更好帮助你看清这些芜杂的。

示例

遗漏的逗号

很多情况下，对象初始值代码会中缺少逗号：

var obj = { 
  a: 1, 
  b: { myProp: 2 }
  c: 3
};

正确的是：

var obj = {
  a: 1,
  b: { myProp: 2 },
  c: 3
};
